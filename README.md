# Quran Player — H-M.github.io

موقع مشغل القرآن الكريم — بث وتحميل السور عبر مصادر مفتوحة (mp3quran.net).

مزايا النسخة الحالية
- عرض 114 سورة مع أسماء بالعربية والإنجليزية (عند التوفر).
- اختيار أي قارئ متوفر عبر API، تشغيل مباشر لكل سورة، روابط تنزيل لكل سورة.
- تحديد سور متعددة وتصدير قائمة تشغيل M3U (حدد/المرئي/الكل).
- إدخال خوادم بديلة (fallback) لاستخدامها عند فشل التحميل.
- واجهة ثنائية اللغة (العربية/الإنجليزية) مع دعم RTL.
- Service Worker بسيط لتخزين واجهة التطبيق (app shell) وتحسين الاعتمادية.

تنصيب وتشغيل محلياً
1. استنسخ المستودع:
   git clone https://github.com/H-M-cmyk/H-M.github.io.git
2. شغّل خادم محلي داخل المجلد:
   - بايثون 3: `python -m http.server 8000`
   - أو Node: `npx http-server -p 8000`
3. افتح المتصفح: http://localhost:8000/index.html

النشر على GitHub Pages
- لأن اسم الريبو بصيغة user.github.io، فعادةً يتم نشر المحتوى من الفرع `main` على المسار root. افتح Settings → Pages → اختر الفرع `main` والمسار `/ (root)`.

المصدر والحقوق
- الملفات الصوتية تُقدَّم عبر مزوّد خارجي (mp3quran.net) أو خوادم بديلة يتيحها المستخدم داخل واجهة الموقع.
- قبل إعادة استضافة أو توزيع ملفات الصوت (مثلاً ZIP للمصحف) — يجب التحقق من شروط استخدام المصدر والحصول على إذن صريح إن لزم.
- هذا المشروع يعرض الروابط والـstream فقط ولا يعيد استضافة الملفات بشكل افتراضى.

اقتراحات مستقبلية
- دعم تنزيل ZIP للمجموعات (يتطلب endpoint سيرفر أو وجود نسخ مستضافة).
- تحسين الوصول (اختبارات a11y، دعم لوحة مفاتيح أعمق).
- تحسين سياسة الكاش في Service Worker والتحكم في ملفات الصوت المخبّأة.
